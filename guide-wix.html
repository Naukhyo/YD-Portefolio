Guide pas-à-pas

Comment reproduire ce portfolio sur Wix

Ce guide explique comment recréer chaque section du portfolio BTS SIO dans l'éditeur Wix Free : quel bloc utiliser, les étapes clés, et à quoi ça ressemble dans Wix.

Avant de commencer — bonnes pratiques Wix

  • Utilise les thèmes de couleur Wix (Paramètres du site → Thème) pour garder une palette cohérente sur toutes les sections.
  • Travaille toujours en mode Desktop d'abord, puis vérifie et ajuste la vue Mobile via l'icône téléphone en haut de l'éditeur.
  • Ajoute des ancres (Ajouter → Menu & Ancres → Ancre) pour que les liens de la nav puissent rediriger vers les sections (Accueil, À propos, Projets, etc.).
  • Uploade toutes tes images dans la médiathèque Wix avant de commencer (Médias → Ajouter des médias). Ça évite les allers-retours pendant le design.
  • Sur Wix Free : pas de domaine personnalisé (ton URL sera nomutilisateur.wixsite.com/ton-site), et une bannière publicitaire Wix apparaît en bas de page — c'est normal, inclus dans la gratuité.
  • Sauvegarde régulièrement avec Ctrl+S. Publie en cliquant sur Publier en haut à droite une fois prêt.

1Header & Navigation

Bloc Wix : Strip + Menu horizontal

Barre supérieure fixe avec le logo à gauche (« YD — Yvan Dupuit ») et les liens de navigation à droite.

Où le trouver

Ajouter (+) → En-tête. Wix ajoute automatiquement un en-tête en haut de chaque page. Pour le menu : Ajouter (+) → Menu & Ancres → Menu horizontal.

Étapes clés

  1. Clique sur l'en-tête par défaut puis Modifier l'en-tête. Choisis un layout avec logo à gauche + menu à droite.
  2. Ajoute ton logo : Ajouter → Texte pour « Yvan Dupuit » ou Ajouter → Image pour un logo « YD » (carré noir 28×28 avec texte blanc).
  3. Ajoute le menu horizontal : Ajouter → Menu & Ancres → Menu horizontal. Glisse-le dans l'en-tête.
  4. Clique sur le menu → Gérer le menu → ajoute les liens : Accueil, À propos, Compétences, Projets, Veille, Contact. Chaque lien pointe vers une ancre de la page.
  5. Dans Paramètres de l'en-tête, active Fixe lors du défilement (sticky) et mets un fond blanc légèrement transparent.

Personnalisation

  • Couleur : fond #FFFFFF (85 % d'opacité), bordure basse #E3E7EE.
  • Police : Inter, taille 14–15 px, poids 500 pour les liens.
  • Survol lien : couleur texte qui s'assombrit + léger fond gris (Interactions → Hover).

2Hero (section d'accueil)

Bloc Wix : Strip avec texte centré + 2 boutons

Grande section d'introduction avec pastille « Disponible pour échanger », titre « Yvan Dupuit », sous-titre, et deux boutons.

Où le trouver

Ajouter (+) → Strip → choisis un layout Title & Subtitle ou Welcome (section plein écran avec texte centré).

Étapes clés

  1. Ajoute un Strip plein largeur, hauteur ≈ 520 px, fond blanc.
  2. Dans le Strip, ajoute un élément Texte (pastille) : « Disponible pour échanger ». Style fond bleu pâle #E8EDFB, texte bleu foncé, coins arrondis 999px.
  3. Ajoute un Titre (H1) : « Yvan Dupuit » — Inter 800, 56 px, letter-spacing négatif.
  4. Ajoute deux paragraphes texte : « Étudiant BTS SIO — Option SISR » et « En alternance chez MP Conception · Lyon / Oyonnax ».
  5. Ajoute deux Boutons (Ajouter → Bouton) :
    • Voir mes projets — style plein, fond bleu #3B5BDB, texte blanc. Lien vers l'ancre #projets.
    • Me contacter — style outline, bordure grise. Lien vers l'ancre #contact.
  6. Centre tous les éléments et ajoute une ancre « accueil » en haut (Ajouter → Menu & Ancres → Ancre).

Personnalisation

  • Couleur accent : #3B5BDB (bleu Indigo).
  • Espacement : padding vertical 96 px (réglable dans Disposition du Strip).

3À propos

Bloc Wix : Strip 2 colonnes (texte / image)

Section présentation sur deux colonnes : texte et cartes d'info à gauche, image pfSense en bas sur toute la largeur.

Où le trouver

Ajouter (+) → Strip → sélectionne un layout 2 colonnes (texte à gauche, conteneur à droite).

Étapes clés

  1. Ajoute un Strip 2 colonnes. Colonne gauche = texte, colonne droite = grille de cartes info.
  2. Colonne gauche : ajoute un éyebrow « À PROPOS » (texte majuscule bleu), un titre « Formation & alternance », puis 2–3 paragraphes sur ton parcours (BTS SIO, SEPR, MP Conception).
  3. Colonne droite : crée 4 petites cartes avec Ajouter → Box → Conteneur. Chaque carte contient un label en capitales (« FORMATION », « ALTERNANCE », « DOMAINE », « OBJECTIF ») et 2 lignes de texte. Dispose-les en grille 2×2.
  4. Image pleine largeur en dessous : Ajouter → Image, sélectionne photo-pfsense.jpg depuis la médiathèque. Largeur 100 %, hauteur max 300 px, coins arrondis 12 px, recadrage Cover.
  5. Ajoute une petite légende italique sous l'image : « Configuration pfSense — Projet BTS SIO ».
  6. Ajoute une ancre « apropos » en haut du Strip.

Personnalisation

  • Cartes info : fond #F7F8FA, bordure #E3E7EE, coins arrondis 10 px, padding 18 px.
  • Image : applique un masque arrondi dans Paramètres image → Bordure & coins.

4Compétences

Bloc Wix : Répéteur (Repeater) ou grille de Cards

Bannière image Proxmox en haut, puis grille 3×2 de cartes « compétences » avec icône + titre + tags.

Où le trouver

Ajouter (+) → Liste → Répéteur. Choisis un layout grille 3 colonnes. Le répéteur permet de dupliquer une carte et de changer juste son contenu.

Étapes clés

  1. Crée un nouveau Strip à fond gris très clair #F7F8FA. Titre « Compétences » + sous-titre.
  2. Bannière image Proxmox : Ajouter → Image, choisis photo-proxmox.jpg. Largeur 100 %, hauteur max 200 px, coins arrondis 12 px, recadrage Cover. Légende italique en dessous : « Installation Proxmox VE — Hyperviseur du projet ».
  3. Sous la bannière, ajoute un Répéteur avec 6 éléments, en grille 3 colonnes.
  4. Dans l'item modèle du répéteur, ajoute : une icône (Ajouter → Icône) dans un petit carré bleu pâle, un titre H3 (Réseau, Systèmes, Virtualisation, Stockage, Sécurité, Outils), puis plusieurs petits « tags » texte (VLAN, pfSense, Cisco, etc.).
  5. Les tags sont de petites Box (Ajouter → Box) avec bordure grise et texte court — duplique-les horizontalement.
  6. Ajoute une ancre « competences » en haut du Strip.

Personnalisation

  • Carte skill : fond blanc, bordure #E3E7EE, coins 16 px, padding 24 px. Au survol → bordure plus foncée + ombre légère (Interactions → Hover).
  • Icône : carré 32×32, fond #E8EDFB, couleur icône #3B5BDB.
  • Tags : fond #F7F8FA, bordure #E3E7EE, texte 12.5 px.

5Projets

Bloc Wix : Répéteur 3 colonnes avec image + texte + bouton

Trois cartes projet côte à côte. Les deux premières sont « featured » avec fond coloré dégradé, la troisième est simple.

Où le trouver

Ajouter (+) → Liste → Répéteur. Choisis un layout 3 colonnes avec image, titre, texte, bouton.

Étapes clés

  1. Nouveau Strip « Projets » fond blanc. Éyebrow « RÉALISATIONS », titre « Projets », sous-titre.
  2. Ajoute un Répéteur 3 colonnes, 3 items.
  3. Item 1 — Active Directory :
    • Badge en haut à droite « Dossier projet 1 » (petite Box bleue).
    • Icône AD (utilisateurs), titre « Active Directory — milkyway.local », description.
    • Galerie 2 images côte à côte : Ajouter → Galerie → Grille, ajoute ad-domaine.png et ad-gpo.png. Hauteur 120 px, coins 8 px.
    • Tags techno (Windows Server 2022, DNS, DHCP, GPO…).
    • Bouton « Voir le dossier → ».
    • Fond dégradé bleu pâle → blanc, bordure bleue.
  4. Item 2 — TrueNAS : même structure, mais en teal (#0E7C7B) au lieu de bleu. 1 seule image : truenas-3.png.
  5. Item 3 — Autres procédures : carte simple, fond blanc, bordure grise, sans image. Juste icône + titre + desc + tags + bouton outline.
  6. Ajoute une ancre « projets ».

Personnalisation

  • Cartes featured : fond linear-gradient(180deg, #E8EDFB, #FFF) pour AD, linear-gradient(180deg, #E6F4F4, #FFF) pour TrueNAS. Dans Wix : Arrière-plan du conteneur → Dégradé.
  • Mini-galerie : si le Répéteur ne supporte pas de galerie imbriquée, remplace par 2 éléments Image côte à côte dans un conteneur grille.
  • Bouton principal : fond accent, texte blanc, coins 9 px.

6Veille technologique

Bloc Wix : Strip texte + icône

Section simple avec une grosse carte : icône ampoule à gauche, tag « Thème actuel », titre, paragraphe à droite.

Où le trouver

Ajouter (+) → Strip → layout Texte avec icône ou crée une simple Box conteneur.

Étapes clés

  1. Nouveau Strip fond gris clair #F7F8FA. Éyebrow « VEILLE TECHNOLOGIQUE » + titre « Sujet en cours ».
  2. Ajoute une grande Box conteneur (Ajouter → Box → Conteneur), fond gris clair, bordure, coins arrondis 16 px, padding 40 px.
  3. À l'intérieur, dispose en 2 colonnes :
    • Gauche : icône ampoule 56×56 dans un carré blanc arrondi (Ajouter → Icône, choisis une ampoule).
    • Droite : petit tag bleu « Thème actuel », titre H3 « L'impact de l'IA sur les métiers de l'informatique », paragraphe descriptif.
  4. Ajoute une ancre « veille ».

Personnalisation

  • Tag bleu : fond #E8EDFB, texte #2D47B3, coins 999 px.
  • Icône : fond blanc, bordure #E3E7EE, coins 12 px.

7Footer / Contact

Bloc Wix : Footer Wix standard

Pied de page avec logo + nom à gauche, email à droite, ligne de copyright en bas.

Où le trouver

Wix a un footer intégré sur chaque page. Clique en bas de ton site → Modifier le pied de page. Ou : Ajouter (+) → Pied de page.

Étapes clés

  1. Choisis un layout de footer simple (2 colonnes + ligne en bas).
  2. Colonne gauche : logo « YD » (petit carré noir 36×36) + texte « Yvan Dupuit » / « BTS SIO — SEPR Lyon ».
  3. Colonne droite : icône enveloppe + lien email yvandupuit@gmail.com. Dans Wix : clique sur le lien → Lien vers → Email, saisis l'adresse.
  4. Ligne du bas : texte centré « © 2025 Yvan Dupuit · Portfolio réalisé dans le cadre du BTS SIO option SISR », séparé par une bordure haute.
  5. Ajoute une ancre « contact » dans le footer pour que le bouton Hero et le lien menu puissent y rediriger.

Personnalisation

  • Fond : #F7F8FA, bordure haute #E3E7EE.
  • Copyright : 12.5 px, couleur #8089A0, centré.
