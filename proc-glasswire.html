GlassWire — Surveillance réseau

Utilisation de GlassWire pour surveiller les connexions réseau d'un poste Windows et détecter les applications suspectes.

← Retour aux procédures

Présentation de GlassWire

GlassWire est un outil de monitoring réseau pour Windows. Il affiche en temps réel toutes les connexions réseau du poste, les applications qui communiquent vers internet, et les volumes de données échangés. Il peut aussi bloquer certaines applications ou alerter en cas de comportement inhabituel.

Interface GlassWire — tableau de bord principal

Liste des connexions actives

L'onglet "Utilisation" liste toutes les applications qui ont eu une activité réseau, avec le volume de données émis et reçu. On peut voir si une application tourne en fond et consomme de la bande passante à notre insu. Pratique pour détecter un logiciel suspect.

Liste des applications avec leur activité réseau

Graphique de trafic

GlassWire affiche un graphique du trafic réseau sur la durée. Les pics de trafic sont visibles et on peut cliquer dessus pour voir quelle application en était responsable. C'est utile pour comprendre une lenteur réseau ou identifier une activité anormale.

GlassWire est un complément à Wireshark : il donne une vue globale et lisible, là où Wireshark permet d'analyser le contenu des paquets. Les deux outils sont complémentaires.
